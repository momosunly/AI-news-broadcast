AI 资讯分类

模型进展

前沿模型发布进入“能力分层 + 访问治理”阶段

GPT-5.6 把 Sol/Terra/Luna 分层，同时对最强能力采用受限访问；这会推动模型公司在产品、价格、安全评估和客户资格上做更细颗粒度管理。

Claude Fable/Mythos 事件成为监管模板

从暂停到解除限制，Anthropic 事件提供了一个新范式：模型公司需要证明主动风险缓解能力，政府则可能引入可信伙伴访问机制。

政策舆情

美国监管从“事后问责”转向“发布前介入”

OpenAI 与 Anthropic 案例说明，网络安全能力已成为前沿模型能否广泛发布的核心门槛。产业界最关心的是评估标准、时限和申诉机制是否透明。

开源模型的地缘政治权重上升

当闭源前沿模型受到访问限制，开源权重与本地部署能力会获得更强扩散优势，也会带来更尖锐的安全争议。

上下游产业

Agent 基础设施竞争转向运行时治理

缓存、评测、追踪、沙箱、文档解析和检索增强开始成为企业导入 Agent 的必需品，而非锦上添花。

算力链继续受数据中心资源约束影响

GPU 供给之外，液冷、电力、水资源、机柜和网络连接会继续决定 AI CapEx 的交付节奏。

下游应用市场

企业文档工作流是近期最明确 ROI

KYC、财务票据、合同审查、法务发现和企业知识库适合用“解析 + 检索 + Agent + 审计轨迹”形成闭环。

AI 工具垂直化继续加速

从 GitHub 社区项目到投资、视频、安全、办公自动化，通用模型正在被封装成可直接执行任务的垂直工具。
